HTML이란?

  1. HTML MDN 설명
  2. HTML은 HyperText Markup Language의 약자로 웹페이지를 만들 때 사용하는 마크업 언어이다.
  3. 웹페이지를 구조화하여 브라우저가 사용자에게 보여주는 화면을 만들 수 있다.
  4. HTML은 태그들로 이루어져 있는데 대표적으로 html, body, div, span, p 태그 등이 있다.
  5. HTML은 어떤 언어들보다 가장 관대한 언어로써, 논리적인 실수 뿐만 아니라 오타와 같은 단순한 오류를 자의적으로 해석하여 나타낸다.

기본 골격을 이루는 태그들

html 태그
웹페이지를 만들 때 가장 먼저 작성하는 태그로, 일반적으로 에디터의 빈 workspace에서 "!" + tap키를 눌러 작성한다. (vscode 기준)
이 경우 html 태그 뿐만 아니라 head, body 태그도 같이 생성되며 손쉽게 html 문서의 골격을 만들어주어 편의성을 제공해준다.
head 태그
head 태그에는 웹페이지의 정보와 관련된 태그들을 작성하며, 외부 css 파일이나 Bootstrap, jQuery 등 외부 프레임워크들을 CDN으로 끌어올 때 head 태그 안에 작성한다.
기본적으로 meta, link, title 태그를 작성하며 웹페이지의 속성, 외부 연결링크, 웹페이지 제목을 작성한다.
body 태그
head 태그에 웹페이지 정보와 관련된 태그를 작성했다면, body 태그에는 본격적으로 웹페이지의 구조를 위한 태그들을 작성한다.
대표적으로 block 속성인 div, p 태그 등이 있으며 inline 속성인 span 태그를 활용할 수 있다.
다만 웹페이지를 구조화할 때 모든 block을 div 태그로 작성하는 것이 아니라 목적에 맞게 태그를 활용해야 개발자와 사용자가 올바르게 이해하고 사용할 수 있다.

head를 채워보자

meta 태그
meta 태그에는 charset에 대한 정보와 브라우저의 viewport를 위한 태그를 작성한다.
charset을 UTF-8로 설정하게 되면 한글을 표시할 때 깨짐없이 사용자에게 보여준다.
viewport는 웹페이지의 화면을 사용자의 모니터에 표시하는 크기를 정의할 수 있다. viewport를 잘 활용하면 반응형 웹페이지를 만드는데 도움이 된다.
meta 태그는 닫힘태그 없이 단독으로 사용한다.
link 태그
link 태그는 외부 스타일시트를 연결해주는 역할을 하고, 일반적으로 별도의 css 파일을 연결할 때 사용한다.
css 문서 뿐만 아니라 웹폰트를 사용할 때 link 태그를 사용하여 연결하고 적용할 범위를 설정하여 사용한다.
link 태그를 사용하여 css 파일을 연결하지 않을 경우 head 태그 내에 style 태그를 작성하여 css를 적용시킬 수 있다.
이 태그 역시 닫힘태그 없이 단독으로 사용한다.
title 태그
웹페이지의 이름을 표시할 때 사용한다. 웹페이지의 이름은 브라우저 상단 탭에 위치한다.
title 태그의 경우 열림태그와 닫힘태그를 사용하여 이름 작성 범위를 나타낸다.
